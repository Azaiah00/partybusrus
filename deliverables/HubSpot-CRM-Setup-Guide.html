Step 2 — Set up your sales pipeline

2

Build the pipeline in HubSpot

  1. In HubSpot, top-right gear icon → Settings
  2. Left menu: Objects → Deals
  3. Click Pipelines tab
  4. Default pipeline exists — rename it "Party Bus Bookings"
  5. Replace default stages with:
    • New Lead (probability 10%)
    • Quoted (probability 30%)
    • Negotiating (probability 60%)
    • Booked — Deposit Paid (probability 100%, won)
    • Lost — Not Booked (probability 0%, lost)
    • Completed (probability 100%, won)
  6. Save

Step 4 — Wire Netlify Forms → HubSpot

Right now your quote form posts to Netlify. We need every submission to ALSO create a contact in HubSpot. There are two ways — pick one.

Option A: Zapier (easiest — recommended for non-developers)

4A

Zapier integration

  1. Create a free Zapier account at zapier.com (100 tasks/month free)
  2. Click Create Zap
  3. Trigger: "Netlify" → "New Form Submission"
    • Connect your Netlify account (it'll walk you through OAuth)
    • Pick the quote-form form
  4. Action: "HubSpot" → "Create or Update Contact"
    • Connect HubSpot account
    • Map fields: Email from Netlify → Email in HubSpot, Phone → Phone, Name → First/Last Name (Zapier can split), event_date → Event Date custom property, etc.
  5. Add a SECOND action: "HubSpot" → "Create Deal"
    • Pipeline: Party Bus Bookings
    • Stage: New Lead
    • Associate with contact created above
    • Deal name: {{event_type}} — {{name}} — {{event_date}}
  6. Turn Zap on. Test by submitting a form.

Option B: Direct webhook (free, slightly more technical)

4B

Netlify Functions → HubSpot API

If Zapier's 100 tasks/mo isn't enough (you'd need 100+ leads/month — good problem to have!), I can build you a Netlify Function that POSTs directly to HubSpot's Forms API. Free forever, no task limits. Let me know if you want this and I'll write the function file.

Roughly: a .js file in /netlify/functions/ that listens for the Netlify form submission webhook, transforms the data, and POSTs to HubSpot's /forms/v2/submissions endpoint with your portal ID + form GUID.

Step 5 — Set up email notifications

You want a notification email the second anyone submits a quote. Two places to set this up:

5

Netlify side — instant notification

  1. Netlify dashboard → your site → Forms
  2. Click the quote-form form
  3. Settings → Form notifications → Add notification → Email
  4. Recipient: info@partybusrus.com (or your personal email until you set up the business email)
  5. Subject: 🚍 New Quote Request from {{name}} — {{event_type}}
  6. Save

HubSpot side — followup reminder

  1. In HubSpot, top-right gear → Settings → Notifications
  2. Enable: "New contact assigned to me", "Deal created", "Deal stage changed"
  3. Email + browser push both on

Step 7 — Set up email marketing lists

Every booked customer becomes a future repeat customer. Every "didn't book" lead becomes someone you can re-engage for prom, holidays, NYE.

7

Build three smart lists

  1. "All Leads" — everyone who ever filled the form
  2. "Booked Customers" — Deal stage = Completed
  3. "Didn't Book — Re-engage" — Deal stage = Lost, marketing_optin = true

HubSpot → Contacts → Lists → Create list → Active list. Set filters by lifecycle stage, deal stage, and properties.

Recurring email campaigns to set up:

  • Monthly newsletter — fleet updates, blog posts, customer spotlights
  • Prom season blast (March/April) — to "Didn't Book" list with prom-specific deal
  • NYE early-bird (October) — book your NYE party bus now
  • Wedding season (February) — booking calendar opening for May-October weddings
  • Birthday auto-trigger — capture birthdays in the form, auto-send "Your birthday's coming up!" email 4 weeks before

Step 8 — Build a dashboard to track leads

8

HubSpot Dashboard

HubSpot → Reports → Dashboards → Create dashboard. Add these tiles:

  • New leads this week — count of contacts created last 7 days
  • Conversion rate — Booked / Total Leads
  • Lead source breakdown — by utm_source property (Google, Instagram, Direct, Referral)
  • Hot leads pipeline — list view of deals in "Quoted" or "Negotiating" stages
  • Upcoming events — sort contacts by event_date asc, show next 30 days
  • Revenue this month — sum of deal value for "Booked" deals this month
